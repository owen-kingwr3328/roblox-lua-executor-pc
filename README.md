## Overview

Roblox Executor is a Windows desktop script executor built for users who want an easy way to run Lua and Luau scripts in Roblox. It emphasizes a lightweight UI, quick access to an integrated script hub, and execution tools that fit naturally into standard Windows usage.

The project targets common Roblox scripting workflows where fast startup, drag-and-drop file loading, and clear execution feedback are useful. It also includes support for multiple Roblox instances, live logs, and a setup that does not depend on extra runtimes, so it is simple to open and run on Windows 10 or Windows 11.
